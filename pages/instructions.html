Instructions for Installing the Email Templates

for Magento 1.9.x.x

  1. Unpack the zip file
  2. Copy the email folder (from Magento1) to 
    app/locale/lang_CODE/template/email
  3. Copy the file "email-inline.css" from css folder to 
    skin/frontend/YOURPACKAGE/YOURTHEME/css/email-inline.css
  4. Make sure to flush all cache
  5. Done!

for Magento 2.x

  1. Unpack the zip file
  2. Copy the "Magento_Email" folder (from Magento2) to 
    app/design/frontend/YOURPACKAGE/YOURTHEME/
  3. Make sure to flush all cache
  4. Done!
